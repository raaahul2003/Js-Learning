-----------

-js is a high-level programming language used to make webpage dynamic
-node.js : provide js runtime environment + js libraries
-basic concepts in js
    -console.log(""): Used to display content
    -data types: 
        >primitive datatype (simple values):
         Number     - any number eg:256, 4.23,785.8754
         String     - text or word in ""
         Boolean    - trur or false
         Undefined  - a variable with no value yet eg: let x // x is not defined
         null       - mean no values but may later assign
         bigint     - used to very large numbers
         symbol     - a unique and immutable value, every symbol is different, even if it has same description



        >non primitive (complex values):
         Array      - list of values
         Object     - collection of data {{key:value} pair}

    -Identifier:  a unique name created by the programmer to define variable , class , functions etc.
        >rules of creating Identifier:
          - can have letters(uppercase,lowercase),digits and underscore
          - the first letter should be either letter or underscore 
          - cannot use keywords as Identifier

    -Keywords: reserved words , eg: if , for , let . var etc 

    -hositing: Hoisiting is a javascripit bahaviour where variables and function deceleration re move  to the top of their scope of during the compilation phase

    -scope: scope refers to where a variable can be accessed in code 
        >Global Scope: a variable declared outside any function or block is in the global scope. it can accesed from anywhere in the program.
        >Function Scope: variable declared inside the function are only accesed within the function.
        >Block Scope: Variables declared inside a block {{}} are only acccesed within the function

    -Variables: used to hold data temporary 
        - variables creation in js

            var variable_name
            variable_name = value  

        >using let Keywords

            let variable_name
